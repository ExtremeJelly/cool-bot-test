## Dependencies

Spiggy uses

- `sqlite3` for its database;
- `fuzzywuzzy` for command autocorrection;
- `matplotlib` for data visualizations; _and_
- `discord.py`, various extensions in `discord.ext`, and `discord-py-slash-command` for its interface.
